Related Docs: object Statement | package api

final case class Statement(markers: Markers, message: Message, arguments: Arguments, throwable: Option[Throwable]) extends Product with Serializable

A statement represents a total logging statement. This is most useful for semantic logging, which converts objects to Statement, but is generally helpful in packaging as well.

markers

the markers associated with a statement.

message

the message

arguments

the arguments

throwable

the throwable, if any.
